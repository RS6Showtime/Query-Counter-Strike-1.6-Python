# Query-Counter-Strike-1.6-Python

## Introduction
A simple script in Python to query Counter-Strike 1.6 Servers

## Usage

```
python main.py --server IP:Port
```

## Scripts
`main.py` represents the script that can query A2S_INFO, A2S_PLAYER, A2S_RULES; the rest of the scripts represent examples of how to query each query.

## Note
`main.py` is not optimized, but you can open an issue if you have some fixes, and I might accept them.
Sometimes it is required to run the script one more time to get the full report.
Used for education only.
